# SportPlanner - Proyecto de Planificación Deportiva Inteligente

## Contexto del Proyecto
SportPlanner es una plataforma avanzada para entrenadores que automatiza la planificación deportiva basada en categorías, niveles y conceptos técnicos/tácticos.

## Arquitectura Técnica
- **Frontend:** Angular 20 (Standalone, SSR, Tailwind CSS 4).
- **Backend:** .NET 10 Web API.
- **Base de Datos & Auth:** Supabase (PostgreSQL).
- **Infraestructura:** Repositorio gestionado con Git.
