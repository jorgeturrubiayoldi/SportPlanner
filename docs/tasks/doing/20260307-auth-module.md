# Task: Implement User Authentication Module

## 📋 Description
Implement a complete authentication system using Angular for the frontend, .NET for the backend API, and Supabase for authentication and storage.

## ✅ Acceptance Criteria
- [ ] Users can sign up and log in.
- [ ] JWT token management integrated between .NET and Supabase.
- [ ] Profile page in Angular showing user data.
- [ ] RLS policies in Supabase to protect user data.

## 🛠️ Execution Plan (Orchestration)
1. **Architect:** Design the database schema and RLS policies.
2. **Backend-Dev:** Implement the .NET API to validate sessions.
3. **Frontend-Dev:** Build the Login and Profile components in Angular + Tailwind.
4. **Tester:** Write unit tests for auth services.
5. **QA:** Verify the complete flow and security.
