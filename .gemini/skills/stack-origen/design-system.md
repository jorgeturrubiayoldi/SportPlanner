# 🎨 Origen Design System (Tailwind)

Use these standards for all UI components to ensure a consistent, modern, and polished look.

## 🌈 Color Palette (Tailwind Classes)
- **Primary:** `indigo-600` (Main actions) / `indigo-700` (Hover)
- **Secondary:** `slate-800` (Sidebars, headers)
- **Background:** `slate-50` (Main background) / `white` (Cards)
- **Success:** `emerald-500`
- **Error:** `rose-500`
- **Text:** `slate-900` (Headers) / `slate-600` (Body)

## 🔘 Components Style
- **Buttons:** `px-4 py-2 rounded-lg font-medium transition-all duration-200 shadow-sm`.
- **Inputs:** `border-slate-200 focus:ring-2 focus:ring-indigo-500 rounded-md`.
- **Cards:** `bg-white border border-slate-100 shadow-sm rounded-xl p-6`.
- **Navigation:** Blur effect (`backdrop-blur-md bg-white/80`) and sticky headers.

## ✨ Visual Polish
- **Borders:** Use `rounded-xl` for a modern "soft" look.
- **Shadows:** Prefer `shadow-sm` for cards and `shadow-md` for floating elements.
- **Icons:** Use Lucide Icons or Heroicons. Always consistent size.
