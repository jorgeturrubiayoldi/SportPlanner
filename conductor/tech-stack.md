# Stack Tecnológico - SportPlanner

## Frontend
- **Framework:** Angular v20.3.0
- **Estilos:** Tailwind CSS v4.1.18 (PostCSS)
- **Estado:** RxJS
- **Internacionalización:** @ngx-translate
- **Integración:** @supabase/supabase-js

## Backend
- **Framework:** .NET 10.0 (ASP.NET Core)
- **Auth:** JWT Bearer (Microsoft.AspNetCore.Authentication.JwtBearer)
- **ORM/DB:** Supabase (C# Client)
- **API Documentation:** OpenAPI (Swagger)

## Base de Datos
- **Motor:** PostgreSQL (vía Supabase)
- **Migraciones:** Scripts SQL manuales en `src/back/SportPlannerNW/Migrations` y archivos con fecha en la raíz del backend.

## Herramientas de Desarrollo
- **Python:** Scripts auxiliares para procesamiento de imágenes y utilidades.
- **Git:** Control de versiones.
